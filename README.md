# GUGUDAN BEAT 🎵

구구단 × 리듬게임. 문제 배너가 내려와 판정선 아래에 자리 잡으면, 박자를 타고 떨어지는 답 노트 중 **정답 숫자만** 골라 치는 낙하 노트 리듬게임입니다.

## 플레이

- **코스 모드**: 단별 암기(정답만 내려옴) → 시험(오답 섞임) → 혼합단 → 혼합 계산(＋ − ×)
- **자유 연습**: 원하는 단만 골라 무한 연습
- 조작: 키보드 `D` `F` `J` `K` 또는 터치/클릭 (떨어지는 숫자를 직접 터치해도 됨)
- 꼬리 달린 **롱노트**는 끝까지 누르고 있으면 보너스
- 틀린 문제는 잠시 후 자동으로 다시 출제 (복습)

## 실행

- 로컬: `게임실행.bat` 더블클릭 (mp3 로드를 위해 로컬 서버 필요)
- 또는 `npx http-server -p 8765` 후 `http://localhost:8765`

## 구성

- `index.html` — 게임 전체 (단일 파일, 의존성 없음)
- `music/` — BGM 트랙 (Suno AI 생성, BPM 자동 분석으로 노트 싱크)
- `sfx/` — 효과음 (ElevenLabs 생성)

음악이 없어도 실행됩니다 — 내장 신스 반주로 폴백.
